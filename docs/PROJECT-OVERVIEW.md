# Luthas.Org — Project Overview

## Summary
Content/blog site — organizational hub, news, projects

## Migration Source
- **WordPress site**: luthas.org
- **Backup source**: UpdraftPlus (Google Drive)
- **Dev domain**: luthas-org.damieus.app

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS + shadcn/ui
- **Architecture**: Feature-Sliced Design (FSD)
- **Deployment**: Vercel
